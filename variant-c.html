A FEATURE ON FUNCTIONAL GRAPHICS

A paint application,
written in OCaml.

And compiled to the browser, entirely. A meditation on what happens when you rebuild a graphical user interface from first principles, one deque at a time.

Before there were frameworks, there were primitives. A function that draws a line. A structure that holds an event. A loop that waits for the mouse. PAINT is a return to that first lineage — a handwritten GUI library and interactive paint application, composed in OCaml and translated, without ceremony, to JavaScript.
